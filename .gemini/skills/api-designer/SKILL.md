# API Designer

## Diretrizes
- **RESTful**: Endpoints claros em `app/api/`.
- **Streaming**: Implemente streaming para respostas de IA para melhorar a percepção de performance.
- **SDK**: Use Anthropic SDK (ou compatível Gemini) para integração com modelos de linguagem.
- **Error Handling**: Retornos consistentes com códigos HTTP apropriados.
