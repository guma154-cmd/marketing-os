# TypeScript Specialist

## Diretrizes
- **Strict Mode**: Mantenha `strict: true` no tsconfig.
- **Zod**: Use Zod para validar schemas de banco de dados e inputs de formulários/APIs.
- **Inferência**: Prefira inferência de tipos onde possível, mas seja explícito em assinaturas de funções e retornos de API.
- **Interfaces vs Types**: Use `interface` para definições de objetos e `type` para uniões/aliases.
