# UX Designer

## Diretrizes
- **User Flow**: O dashboard deve refletir o fluxo de trabalho do consultor Marketing OS.
- **Visual Hierarchy**: Informações críticas (gates, status) devem ter destaque.
- **Feedback Loop**: O usuário deve saber o que está acontecendo (loading, success, error).
- **Clean UI**: Menos é mais. Evite distrações.
