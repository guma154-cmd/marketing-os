# Marketing OS - Framework de Marketing para PMEs

Repositório operacional para gestão de marketing estruturado em 5 fases.

## Fases do Projeto
- **Fase 1: Fundação** -> Estrutura base, configuração de ambiente e definições iniciais.
- **Fase 2: Estrutura Operacional** -> Onboarding de clientes, briefings e diagnósticos.
- **Fase 3: Operação por Cliente** -> Gestão de campanhas, produção de conteúdo e automações.
- **Fase 4: Escala e Hardening** -> Consolidação de processos, logs de escala e auditoria.
- **Fase 5: Execução Comercial** -> Foco em fechamento e crescimento contínuo.

## Estrutura do Repositório
- `docs/`: Artefatos centrais e stories de clientes.
- `evals/`: Verificações de integridade (Ralph Wiggum Loop).
- `.aiox-core/`: Core do framework operacional.
