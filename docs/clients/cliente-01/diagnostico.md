## 4. Matriz de Diagnóstico

### 4.1. Dores Visíveis (O que o cliente sente)
*   "Não tenho tempo para postar."
*   "Sinto que meu perfil não passa profissionalismo."
*   "Gostaria de atrair clientes mais lucrativos."

### 4.2. Dores Ocultas / Gargalos Reais (O que identificamos)
*   **Ausência de Funil Base:** Não há um destino claro para quem chega no perfil.
*   **Ativos Desperdiçados:** Provas sociais valiosas estão "escondidas" em vez de serem usadas na linha editorial.
*   **Incerteza de Avatar:** O cliente não sabe quem é seu público mais lucrativo (PENDENTE de definição no briefing).

### 4.3. Oportunidades (Ganhos Rápidos)
*   **Otimização de Bio/CTA:** Resultados imediatos em conversão de cliques para o WhatsApp.
*   **Uso de Acervo:** Publicar os depoimentos já existentes para gerar autoridade imediata.
*   **Nicho pouco explorado:** Se bem posicionado, o cliente pode se tornar a autoridade local rapidamente.

### 4.4. Riscos Operacionais
*   **Bloqueio por Inexperiência:** Risco de tentar rodar anúncios sem Pixel ou infraestrutura adequada.
*   **Expectativa vs. Verba:** Risco de frustração se o objetivo for alto e a verba de mídia (ainda pendente) for baixa.
