# User Story 1.5: Template de Bio + CTA Base

**Como** profissional do Marketing OS,
**Quero** um template padronizado de Bio + CTA,
**Para** transformar perfis de clientes em ferramentas de conversão claras e profissionais, garantindo uma primeira impressão impactante e facilitando o contato imediato.

## Descrição
A **Bio e o CTA** são os elementos de "vitrine" e "balcão" de um perfil profissional. No método Marketing OS, esta é a Etapa 3 (Estrutura de Perfil). Esta story visa criar o arquivo `docs/templates/bio-cta-base.md`, que fornecerá as fórmulas e diretrizes para reestruturar a bio, o nome do perfil, os destaques estratégicos e o link de contato, consumindo o posicionamento definido nas etapas anteriores de briefing e diagnóstico.

## Critérios de Aceitação

### 1. Criação e Governança do Template
- [x] Criar o arquivo `docs/templates/bio-cta-base.md`.
- [x] Incluir instruções de reutilização e personalização para novos clientes e diferentes nichos.
- [x] Garantir que o template seja prático e visualmente fácil de seguir pelo consultor.

### 2. Estrutura de Bio Profissional (Conversão e SEO)
- [x] Definir diretrizes para o **Nome do Perfil**: Uso obrigatório de [Nicho + Nome/Marca] para indexação de busca.
- [x] Estruturar a **Bio** em 3 linhas estratégicas:
  - **Linha 1: Promessa:** O que você resolve e para quem (Proposta de Valor).
  - **Linha 2: Autoridade/Prova Social:** Uso de números, tempo de mercado ou conquista relevante.
  - **Linha 3: Chamada para Ação (CTA):** Instrução clara do próximo passo.
- [x] Criar uma biblioteca de variações de CTA (ex: "Agende sua avaliação", "Peça seu orçamento", "Fale com nossa equipe").

### 3. Diretrizes de Conversão Complementar
- [x] Definir a estrutura recomendada para **Destaques Estratégicos**: Sugestão de 4 a 5 destaques essenciais (Comece Aqui/Serviços/Resultados/FAQ/Local).
- [x] Fornecer orientações para o **Link na Bio**: WhatsApp direto (gerador de link com mensagem) ou ferramenta de links múltiplos.
- [x] Incluir modelos de **Mensagem de Entrada** automática para o WhatsApp, personalizadas por tipo de CTA.

### 4. Conectividade com o Método
- [x] Validar que o template utilize as definições de "Oferta Principal" e "Diferencial" levantadas no `diagnostico-base.md`.
- [x] Garantir linguagem focada em clareza, confiança e ação.
