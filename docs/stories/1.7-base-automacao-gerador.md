# User Story 1.7: Base para Automação / Gerador de Conteúdo

**Como** consultor do Marketing OS,
**Quero** um template que normalize os dados estratégicos do cliente para automação,
**Para** gerar ideias, legendas, CTAs e hashtags de forma consistente, rápida e comercialmente orientada, evitando conteúdos genéricos e sem propósito.

## Descrição
A **Base para Automação** é o elo entre a inteligência estratégica (Briefing/Diagnóstico) e a execução em escala (Conteúdo). Esta story visa criar o arquivo `docs/templates/base-automacao-gerador.md`, que funcionará como um "tradutor de contexto" para modelos de linguagem (LLMs). O objetivo é garantir que a IA entenda exatamente quem é o cliente, o que ele vende e para quem, gerando saídas que respeitem o posicionamento e a autoridade da marca.
