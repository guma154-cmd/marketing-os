# User Stories (Backlog)

Esta pasta contém o histórico e o planejamento das atividades do projeto Marketing OS, organizadas por Epics e Stories.

## Status das Stories
- `backlog`: Planejada, mas sem detalhamento.
- `ready`: Refinada e pronta para execução.
- `in_progress`: Sendo executada.
- `ready_for_review`: Aguardando validação do PO.
- `approved`: Validada pelo PO.
- `closed`: Concluída e arquivada.
