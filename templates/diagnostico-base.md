## 4. Matriz de Diagnóstico

### 4.1. Dores Visíveis (O que o cliente sente)
*Problemas relatados pelo cliente (ex: "não tenho tempo para postar", "meu engajamento caiu").*

### 4.2. Dores Ocultas / Gargalos Reais (O que o consultor identificou)
*Problemas técnicos que o cliente não percebe (ex: Bio sem palavra-chave, falta de oferta direta, tráfego sem pixel configurado).*

### 4.3. Oportunidades (Potencial de ganho rápido)
*Ativos que o cliente já tem mas não usa bem (ex: muitos seguidores inativos, lista de ex-clientes, ótimas fotos guardadas).*

### 4.4. Riscos Operacionais
*Fatores que podem travar o projeto (ex: verba muito baixa para o objetivo, histórico de bloqueios no Facebook, falta de braço para atendimento).*
